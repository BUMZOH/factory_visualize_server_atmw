# 従来のアプリ（光和業務で使用するアプリ）

### 光和業務アプリランチャ\mw_app_launcher.py
➡ 光和用に作成した種々のアプリを起動するアプリ

### KV5000稼働データ受信/receive_operation_data.py
➡ 中央監視PLCの内部デバイスに格納されているデータを受信し、Excelファイルに貼り付ける  
　 なお、SDカード中のCSVファイルではない。  
　 夜中の1時にタスクスケジューラにより自動起動される。  

### KV5000稼働データ受信/show_detail.py
➡ 稼働データが保管されているExcelファイルで選択されたセルに対応する設備、日付の詳細データを表示する。  
　 稼働データ一覧表と帯グラフにより表示。

### VT異常履歴取得PG/GetAlmdataFromVT.py
➡ タッチパネル編集ソフト(VT Designer)からLAN上のタッチパネルにアクセスし、アラーム履歴をPCへ保存する。
　 USB-Ethernet変換装置を利用している(VT3が認識に時間が掛かり不安定)。  
　 また、PyAutoguiを利用した自働化であり、かなり強引な自働化手法。  

### チョコ停自動集計PG/AlmAggregation.py
➡ VTのアラーム履歴データを集計し、ランキング表示、棒グラフにより見える化する。
